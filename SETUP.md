# 企业微信后台配置手册

本文档指导你在企业微信管理后台完成配置，获取所有必要凭证。

---

## 第一步：登录企业微信管理后台

访问 [work.weixin.qq.com](https://work.weixin.qq.com) → 登录企业账号（需有管理员权限）。

---

## 第二步：获取企业 CorpID

1. 左侧导航 → **我的企业** → **企业信息**
2. 页面底部找到 **企业ID（CorpID）**，复制到 `.env` 的 `CORP_ID`

---

## 第三步：创建微信客服账号，获取 open_kfid

1. 左侧导航 → **客户联系** → **微信客服**
2. 点击 **新建客服账号**，填写名称和头像
3. 创建后，在账号列表中找到该账号，点击查看详情
4. 复制 **客服账号ID**（格式：`wkXXXXXXXXXXXXXX`）→ 填入 `.env` 的 `OPEN_KF_ID`

---

## 第四步：创建自建应用，获取 AgentID 和 Secret

1. 左侧导航 → **应用管理** → **自建** → **创建应用**
2. 上传 Logo，填写应用名称，选择可见范围
3. 创建后，进入应用详情页：
   - 复制 **AgentId** → 填入 `.env` 的 `AGENT_ID`
   - 点击 **Secret** 旁的查看（会发送到企业微信 App）→ 填入 `.env` 的 `CORP_SECRET`

---

## 第五步：配置消息回调

1. 在第四步的应用详情页，找到 **接收消息** 区域，点击 **设置API接收**
2. 填写以下信息：
   - **URL**：`https://你的域名/callback`（本地调试用 ngrok：`https://xxx.ngrok-free.app/callback`）
   - **Token**：自定义任意字符串 → 同时填入 `.env` 的 `TOKEN`
   - **EncodingAESKey**：点击 **随机生成**，复制 43 位字符 → 填入 `.env` 的 `ENCODING_AES_KEY`
3. 点击 **保存** 前，确保你的服务已启动（企业微信会发 GET 请求验证）

---

## 第六步：授予应用权限

1. 进入应用详情 → **企业微信API** → **通讯录** → 根据需要授权
2. 重要：需要开通 **客服消息** 读写权限（客户联系 → 权限管理）

---

## 第七步：本地调试（ngrok）

```bash
# 安装 ngrok（如未安装）
brew install ngrok

# 启动 ngrok，暴露本地 8000 端口
ngrok http 8000

# 复制输出的 https 地址，如：
# https://abc123.ngrok-free.app
# 将 https://abc123.ngrok-free.app/callback 填入企业微信后台
```

> 注意：免费版 ngrok 每次重启地址会变，需重新在企业微信后台更新 URL。

---

## 第八步：验证配置

1. 确认 `.env` 所有字段已填写
2. 启动服务：`uvicorn main:app --reload --port 8000`
3. 在企业微信管理后台点击 **保存**，若显示"验证成功"则配置完成
4. 用微信扫描客服账号的二维码，发送一条消息，观察服务日志

---

## 常见问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| URL 验证失败 | 服务未启动或地址不可达 | 确认 ngrok 在运行，URL 正确 |
| 消息无回复 | Token/Key 不匹配 | 对照 `.env` 和后台配置，确保一致 |
| 发送失败 401 | access_token 过期或 Secret 错误 | 重新检查 CORP_SECRET，确认应用权限 |
| 收不到消息 | 未授权客服消息权限 | 检查第六步权限配置 |
